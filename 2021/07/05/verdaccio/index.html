cluster verdaccio npm私服 2021/07/05

公司vue项目越来越多，对node项目的支撑，迫切需要npm私服的支持。
所以搭建npm私服，行业verdaccio 是最佳选择，所以自己搭建了一个私服。
因为是新申请主机，我把从零开始搭建一个私服过程记录下。

1、 verdaccio 首先需要node私服支持，所以先安装环境。

npm node环境安装：

下载安装包：
https://nodejs.org/dist/v14.17.2/node-v14.17.2-linux-x64.tar.xz

配置node环境变量:

查看版本：

2. verdaccio 安装:

2.1 全局安装verdaccio：

为了保证使用过程中，管理和稳定性。
全局安装pm2，用来守护node进程：

pm2 start ** // 启动;
pm2 reload ** //重启 ;
pm2 logs *** // 日志

2.2 配置:
这个配置：/opt/node/node/lib/node_modules/verdaccio/conf/default.yaml

新增监听端口:

node: default.yaml ，这个是初始配置，实际使用这个配置会被拷贝到，运行用户配种环境中。

如果有其他各种管理，基本都在这个配置中。

2.3 由于npm 私服不建议使用root 用户：

新建用户:
verdaccio

默认的配置，随意的注册用户。但是对于公司私服来说，通常不希望这样。
