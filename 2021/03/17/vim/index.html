os os vim 2021/03/17

vim介绍

能强大、高度可定制的文本编辑器，曾经Slashdot Beanie的最佳开放源代码文本编辑器大奖。
在linux下掌握一项强大的编辑器对我们也是很有必要的，所以推荐大家好好使用下。

1 vim 工作模式

模式介绍 进入方法 说明
命令行模式 不管用户处于何种模式，只要按一下ESC键 命令行
文本输入模式 命令模式下输入 i、a、o、c、r 需要编辑到
末行模式 在命令模式下，用户按”:”” 末行模式 可以设置 显示行数等操作等

2 常用命令

常用类操作 操作方法 功能和用途
进入编辑 i 插入光标前一个字符
进入编辑 I 插入行首
进入编辑 a 插入光标后一个字符
进入编辑 A 插入行未
进入编辑 o 向下新开一行,插入行首
进入编辑 O 向上新开一行,插入行首
移动光标操作 h 左移
移动光标操作 j 下移
移动光标操作 k 上移
移动光标操作 l 右移
移动光标操作 M 光标移动到中间行
移动光标操作 L 光标移动到屏幕最后一行行首
移动光标操作 G 移动到指定行,行号 -G
移动光标操作 w 向后一次移动一个字
移动光标操作 b 向前一次移动一个字
移动光标操作 { 按段移动,上移
移动光标操作 } 按段移动,下移
移动光标操作 Ctr-d 向下翻半屏
移动光标操作 Ctr-u 向上翻半屏
移动光标操作 Ctr-f 向下翻一屏
移动光标操作 Ctr-b 向上翻一屏
移动光标操作 gg 光标移动文件开头
移动光标操作 G 光标移动到文件末尾
删除相关操作 x 删除光标后一个字符,相当于
删除相关操作 Del X: 删除光标前一个字符,相当于 Backspace
删除相关操作 dd 删除光标所在行,n dd 删除指定的行数
删除相关操作 D 删除光标后本行所有内容,包含光标所在字符
删除相关操作 d0 删除光标前本行所有内容,不包含光标所在字符
删除相关操作 dw 删除光标开始位置的字,包含光标所在字符
撤销命令 u 一步一步撤销
撤销命令 Ctr-r 反撤销
重复命令 . 重复上一次操作的命令
重复命令 n 重复某个命令 n 次，
复制粘贴 yy 复制当前行,
复制粘贴 n yy 复制 n 行
复制粘贴 p 在光标所在位置向下新开辟一行,粘贴
可视模式 v 按字符移动,选中文本
可视模式 V 按行移动,选中文本可视模式可以配合
可视模式 d, y, >>, << 实现对文本块的删除,复制,左右移动
替换操作 / str查找
替换操作 n 下一个
替换操作 N 上一个
替换命令 :%s/abc/123/g 把abc全部替换成123
替换命令 :1, 10s/abc/123/g 将第一行至第10行之间的abc替换成123
